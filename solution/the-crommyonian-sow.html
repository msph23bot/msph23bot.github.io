Solution: The Crommyonian Sow
Answer: STATELY

Written by Jonah Ostroff

This is one of two metas in the Classics round, and solvers must determine which four of the eight answers it uses.

The Crommyonian Sow says you should be learning its language instead. It’s a pig, and so this language is Pig Latin. This is the “Latin” meta, and its answers are:

BEAU
PLUNDER
RUST
VIN

Each of these answers can be translated into Pig Latin to create a new word:

OBEY
UNDERPLAY
ASTRAY
INVEIGH

These all start with different vowels, so we can associate the cells labeled A, I, O, and U in the grid with these new words, and index into them by the numbers.

The grid consists of a bunch of horizontal, vertical, and diagonal dotted lines. We can fill in the indexed letters from the answers using the Crommyonian Sow’s written language: pigpen. Some squares are left blank, if their indices are too high or if the given vowel is E.

Doing so spells the answer, STATELY.
