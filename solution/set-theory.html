Each set of numbers solves in a different way to a 3-digit number.

Subpuzzle 1

Pair a positive number with a negative number and add them together. The new numbers are composed of X followed by N zeroes, followed by Y (and maybe some more zeroes). X & Y are all between 1 & 26, single digit numbers preceded by a 0. N is in increasing order (1-5). Order by N and read X/Y as A1-Z26.

SI: 190090000 = 409818765 - 219728765
XO: 240015000 = 501226345 - 261211345
NE: 140000500 = 374201122 - 234200622
FO:  60000150 = 421010971 - 361010821
UR: 210000018 = 626666659 - 416666641

For an answer of 614.
